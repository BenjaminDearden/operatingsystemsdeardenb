To build and run this program, launch a terminal window in the same path as the source file.
run the following commands: 
cmake ../ProgramOneC++
cmake --build .

After those two commands you should be able to 
enter the program name to launch it with the name of the file
to open listed after the program name 
Usage Examples:
./ProgramOneC++ <filenamehere>

./ProgramOneC++ file1.txt

If it does not build due a directory mismatch delete the CMakeChace.txt file and retry
